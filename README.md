Funcionalidades
🔐 Sistema de login administrativo
👤 Cadastro de clientes com validações:
Nome (formatação automática e validação)
CPF (validação básica e cálculo dos dígitos verificadores)
Idade
📋 Listagem de clientes ativos
🔍 Busca de clientes:
Por nome
Por CPF
✏️ Alteração de dados:
Nome
CPF
Idade
Login
Senha
🔄 Ativar/Desativar usuário
🎨 Interface em terminal com cores e navegação por menu
🧠 Conceitos aplicados
Manipulação de listas e dicionários
Validação de dados (CPF e entrada do usuário)
Estruturas de repetição (while, for)
Estruturas condicionais
Modularização com funções
Simulação de CRUD (Create, Read, Update, Delete)
Tratamento de entrada do usuário
⚠️ Observações
Os dados são armazenados em memória (lista), ou seja, são perdidos ao encerrar o programa
O sistema foi desenvolvido com foco educacional, visando prática de lógica e estrutura de dados
Há partes ainda em desenvolvimento (como alteração de senha no menu inicial)
